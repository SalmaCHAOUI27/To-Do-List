# 📔 To-Do List App with Flask

A simple and elegant To-Do List Web App built using:

- 💻 HTML, CSS, JavaScript
- 🐍 Python + Flask (back-end)
- 🗃️ SQLite (database)

## ✨ Features

- Add tasks with a deadline 📅
- Delete tasks ❌
- Store data in a local database (SQLite)
- Responsive & modern UI
